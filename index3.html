Motion Equations - Interactive Lesson

The kinematic equations describe the motion of objects under constant acceleration. These equations are fundamental to solving physics problems.

The Four Kinematic Equations:

v = v₀ + at

x = x₀ + v₀t + ½at²

v² = v₀² + 2a(x - x₀)

x - x₀ = ½(v₀ + v)t

Variables:

  • v: final velocity
  • v₀: initial velocity
  • a: acceleration
  • t: time
  • x: final position
  • x₀: initial position

Quick Quiz

Which equation would you use to find the final velocity if you know the initial velocity, acceleration, and displacement (but not time)?

v = v₀ + at
x = x₀ + v₀t + ½at²
v² = v₀² + 2a(x - x₀)
x - x₀ = ½(v₀ + v)t

Correct! The equation v² = v₀² + 2a(x - x₀) allows you to find the final velocity without knowing the time.
